### 添加新产品
在 `script.js` 文件的 `products` 数组中添加新产品：

```javascript
{
    id: 13,
    name: "新产品名称",
    description: "产品描述（可参考小红书内容）",
    price: 价格,
    originalPrice: 原价,
    image: "images/brush-xx.JPG",
    category: "类别",
    badge: "标签"
}
```
